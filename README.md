## Description du projet

AfriConnect Summit 2026 est un site web vitrine présentant un sommet africain fictif consacré à la technologie, au business et à l'innovation.

Le site est composé de quatre pages :
- Accueil
- Programme
- Intervenants
- Contact

L'objectif est de proposer une interface moderne, responsive et interactive grâce à JavaScript.

## Fonctionnalités JavaScript implémentées

- Mode sombre avec mémorisation du choix de l'utilisateur.
- Menu burger responsive pour les petits écrans.
- Changement d'apparence du header lors du défilement.
- Bouton "Retour en haut" avec défilement fluide.
- Compte à rebours jusqu'au début de l'événement.
- Animation des compteurs au défilement.
- Apparition progressive des éléments au scroll.
- Navigation par onglets sur la page Programme.
- Filtrage des intervenants par catégorie.
- Validation du formulaire de contact.
- Mise à jour automatique de l'année dans le footer.
